libcrawl

Web crawling framework

Synopsis

cc flag file … -lcrawl library …

#include <libcrawl.h>

Description

libcrawl is a library for constructing web crawlers by implementing the basic logic flow using an application-provided queue implementation and providing hook points at different stages of the crawl process.

Interfaces

libcrawl provides the public interfaces listed below.

FunctionDescription
crawl_createCreate a crawling context
crawl_destroyDestroy a crawling context
crawl_cache_schemeObtain a registered cache implementation for a given URI scheme
crawl_set_cacheSet the cache implementation which will be used by a crawling context
crawl_set_acceptSpecify the Accept header to be used in future requests
crawl_set_uaSpecify the User-Agent header to be used in future requests
crawl_set_userdataSet an application-specific data pointer in a crawling context
crawl_set_verboseSpecify whether network requests will happen verbosely
crawl_set_cache_pathSet the cache filesystem path or URI string
crawl_set_cache_uriSet the cache URI
crawl_userdataObtain the application-specific data pointer
crawl_set_usernameSet the username (or access key) to be used by the cache implementation
crawl_set_passwordSet the password (or secret) to be used by the cache implementation
crawl_set_endpointSet the endpoint to be used by the cache implementation
crawl_set_uri_policySet the URI policy callback
crawl_set_updatedSet the resource-updated callback
crawl_set_failedSet the request-failed callback
crawl_set_nextSet the queue-next callback
crawl_set_checkpointSet the checkpoint callback
crawl_set_unchangedSet the resource-unchanged callback
crawl_set_prefetchSet the prefetch callback
crawl_set_loggerSet the logging callback

Dependencies

libcrawl depends upon libcurl, Jansson, and liburi.
